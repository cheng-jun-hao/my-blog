# CJH 的技术博客 — 设计文档

## 概述

为 CJH 搭建一个纯静态个人博客网站，部署到 GitHub Pages。网站采用简约白净风格，展示 CSDN 博客文章链接、个人简介与项目经历、联系方式。

## 技术方案

纯静态 HTML/CSS/JS，零依赖，无需构建工具。文章数据以 JSON 数组存储在 JS 文件中，后续新增文章只需添加一条记录。

## 文件结构

```
├── index.html          # 主页面（单页应用）
├── css/
│   └── style.css       # 样式文件
├── js/
│   ├── main.js         # 主逻辑（导航、滚动、交互）
│   └── articles.js     # 文章数据与渲染逻辑
└── assets/
    └── (头像等图片资源)
```

## 页面结构

单页应用，3 个区域通过顶部导航切换，点击平滑滚动到对应区域。

### 1. 首页 / 博客文章列表

- 顶部固定分类标签栏：全部 | 网络安全 | AI | 前端开发 | 系统工具 | 其他
- 点击标签筛选对应文章，带淡入动画
- 文章卡片：白色卡片 + 轻微阴影，hover 时微微上浮
- 每张卡片显示：标题、发布日期、分类标签、AI 撰写的精炼摘要、"阅读全文 →" 链接到 CSDN 原文
- 按时间倒序排列

## 视觉设计

### 配色

| 用途 | 色值 |
|------|------|
| 主背景 | #ffffff |
| 次背景（卡片、分隔） | #f8f9fa |
| 主文字 | #1a1a1a |
| 次文字 | #6c757d |
| 强调色（链接、标签、按钮） | #2563eb |
| 边框 | #e5e7eb |

### 排版

- 字体：系统无衬线字体栈（-apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif）
- 正文字号：16px，行高 1.8
- 标题：h1 2rem，h2 1.5rem，h3 1.25rem

### 响应式

- 适配桌面和移动端
- 移动端导航折叠为汉堡菜单

## 文章数据

存储在 `js/articles.js` 中，每条记录格式：

```js
{
  title: "文章标题",
  date: "2025-12-23",
  category: "网络安全",
  summary: "AI撰写的精炼摘要",
  url: "https://blog.csdn.net/qq_68725483/article/details/xxx"
}
```
